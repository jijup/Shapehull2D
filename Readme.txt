Operating system:- Ubuntu 14.04 or later
The program uses CGAL and is tested in CGAL 4.6.

1. Execute the command "sh script.sh"
   Which will:
       a. Download required softwares including cmake, CGAL, boost, glut etc.
       b. Add a flag variable to "CGAL\Point_3.h"
       c. Run cmake and make commands
2. To execute the program:
   
  ./TestShapeHull filename.txt
 
 3. Run the program. 

 4. Provide the display size of vertices. Give "1", if it is not relevant to you.
